Overview

A COS 484 (NLP) final project with co-authors Faith Best and Kaitlyn Wen. We evaluated whether a smaller open-weight model (Llama-3.1-8B-Instruct) fine-tuned with LoRA could outperform GPT-4o with RAG on legal question answering in terms of reliability — using the 2025 U.S. House Oversight Epstein Estate documents (2.1M rows, 25,000+ files) as the test corpus.

Approach

Built a 100-question benchmark stratified across four difficulty tiers: Direct, Multi-Document, Redacted, and Trick questions
Developed a granular error taxonomy (Fabrication, Misattribution, False Legal Inference, Over-Abstention, Chunk Boundary) to profile failure modes qualitatively
Fine-tuned Llama-3.1-8B-Instruct with LoRA (rank 32, lr 5×10⁻⁵, 1 epoch) on 25,000 documents to avoid overwriting Meta's RLHF pretraining
RAG pipeline: BM25 retrieval over 2.1M rows, reranked to top-k, with a confidence-gated forced abstention mechanism
Core finding: fine-tuning creates a "confidence trap" — the model shifts from cautious over-abstention to authoritative fabrication, suppressing epistemic humility

Overview

An independent work report (Spring 2026) advised by Kyle Jamieson. Break Point classifies breakdancing moves in a set using two complementary modalities: computer vision via MediaPipe landmark extraction, and Sony mocopi IMU sensors (12-sensor Pro Kit, 60 Hz). The original goal was to compare novice and seasoned breakers, but the project pivoted to move classification following IRB delays. Fifteen move classes were defined across three categories: Toprock (Open Step, Cross Step, Kick Step, Kickcross Step, Shuffle), Floorwork (CC, Hook, Sweep, 6-Step, 3-Step, Short Drop, Long Drop), and Freezes (Standing Arms Open, Standing Arms Closed, Baby Freeze, Turtle Freeze).

Approach

Skill Issue Jan 2026

Full-stack platform that analyzes player habits and weaknesses across hundreds of chess matches. Uses HDBSCAN clustering and LASSO regression to extract mistake patterns from feature vectors of historical game data, then delivers personalized AI-generated feedback.

Python React + Vite PostgreSQL TailwindCSS AWS HDBSCAN OpenAI API

Overview

An independent work report (Fall 2025) advised by Robert Fish. Skill Issue is a web app that goes beyond Chess.com's "Insights" feature by identifying game-state-specific bad habits — not just raw mistake percentages. Users sign in with Google, link their Chess.com account, and the pipeline automatically pulls PGN game files, runs Stockfish centipawn loss analysis, encodes board positions as feature vectors, clusters them with HDBSCAN, finds contextual correlations via LASSO regression, and generates personalized LLM feedback. Targets the $3.34B global chess market and 1.5M+ Chess.com premium subscribers.

Pipeline

Step 1 — Move Analysis: Fetch PGN files from Chess.com API, replay games move-by-move, evaluate each position with Stockfish (centipawn loss), classify mistake severity
Step 2 — Feature Vectors: Encode board state per mistake move (piece mobility, king safety, pawn structure, castling state, material balance, etc.) into dense feature vectors
Step 3 — Habit Detection: HDBSCAN clusters mistake vectors across hundreds of games; LASSO regression identifies which board features most predict each cluster (contextual habit)
Step 4 — Feedback Generation: LLM (OpenAI) prompted with cluster context and LASSO-identified features to generate specific, actionable improvement advice
Full-stack: React + Vite frontend, Python Flask API, PostgreSQL on AWS — designed as freemium SaaS with limited free tier and paid habit analysis

Mindset & Volatility Apr 2026 – May 2026

Modeled fixed vs. growth mindset as a static parameter in a three-level Hierarchical Gaussian Filter (HGF) to study how each mindset adapts to volatile environments. Growth mindset agents showed up to 266% higher dynamic learning rates in highly volatile conditions, with significantly better accuracy.

DLR +266% in volatile phase accuracy +18.8% volatile early parameter recovery r=0.979
Python Bayesian Inference HGF Computational Psychiatry R

Overview

A PSY 360 (Computational Cognitive Science) final project advised by Professor Brenden Lake. The project asks whether Carol Dweck's fixed and growth mindset theory can be formalized as a computational model of cognition. Using a three-level Hierarchical Gaussian Filter — a Bayesian framework for modeling belief updates under uncertainty — mindset is encoded as a static volatility parameter (θ₃) that governs how an agent weighs new information against its prior beliefs. Agents then complete a 300-trial reward task with three distinct phases: a stable phase, a reversed stable phase, and a volatile phase in which the reward probability flips every 3–6 trials.

Key Results

+266%
Growth vs. fixed mindset dynamic learning rate in the late volatile phase
+18.8%
Accuracy advantage for growth mindset in the early volatile phase (p < 10⁻¹²)
0.979
Pearson correlation for parameter recovery — confirms model identifiability
3-level
HGF hierarchy: observation, volatility estimate, and meta-volatility

Approach

Implemented the full HGF update equations (Mathys et al., 2014) from scratch: precision-weighted prediction errors propagate upward through three hierarchical levels at each trial
Mindset encoded as θ₃ — the meta-volatility prior. Fixed mindset (θ₃ = 1.5) assumes a stable world; growth mindset (θ₃ = −1.5) assumes the world is more changeable, yielding a higher dynamic learning rate
300-trial binary reward task: stable (trials 1–80, Pr[reward] = 0.8), reversed stable (81–160, Pr[reward] = 0.2), and volatile (161–300, probability flips randomly every 3–6 trials)
Statistical analysis via t-tests on dynamic learning rate and accuracy across all three blocks; growth mindset agents significantly outperform in both learning rate and accuracy during volatile conditions
Parameter recovery confirms the model is identifiable: simulated parameters can be reliably recovered from generated behavior (Pearson r = 0.979, MAE = 0.582)

Fully browser-native — hosted on GitHub Pages with no backend, runs purely in WebGL via Three.js

About the Lab

The Villarini Research Group at Princeton's High Meadows Environmental Institute (HMEI) studies flood hydrology, extreme weather events, hydroclimatology, and seasonal climate prediction. Led by Professor Gabriele Villarini (CEE & HMEI), the group builds statistical-dynamical forecasting systems that integrate outputs from global climate models to predict North Atlantic tropical cyclone activity months in advance.

What I Built

Multi-model forecasting system: Integrated sea surface temperature (SST) outputs from 11 operational climate centers — ECMWF, NMME NASA, NMME NCEP, BOM, CMCC, DWD, ECCC, JMA, Météo-France, NCEP, and the UK Met Office — via the Copernicus Climate Data Store (CDS) API and NMME portal
Statistical regression models: Applied Poisson regression to predict tropical cyclone (TC) and hurricane (HU) counts, and Gamma regression to predict Power Dissipation Index (PDI) and Accumulated Cyclone Energy (ACE), using North Atlantic and tropical SST anomalies (SSTMDR and SSTTROP) as predictors
Multi-Model Ensemble (MME): Built three ensemble weighting schemes — equal-weighted average, RMSE-weighted average, and hierarchical — that users can switch between or manually configure per-model weights on the site
Automated forecast verification: Implemented CRPS, Brier Score, Decomposed MSE Skill Score, ROC curves, PIT histograms, reliability diagrams, and 3×3 contingency tables via Pandas and SciPy — enabling systematic comparison of model predictive accuracy across initialization months going back to 1952
Automated data pipeline: Wrote Selenium scrapers and CDS API wrappers to auto-ingest monthly SST forecasts, HURDAT2 storm records (via the Tropycal API), and HadISST1 observed SST data on rolling schedules throughout the hurricane season (Aug–Oct)
Full-stack website: Built the researcher-facing interface with Flask and PostgreSQL — dynamic time-series charts, interactive ensemble configuration, and on-demand retrospective forecast generation. Validated against HURDAT2 observed data from 1952 to 2024
